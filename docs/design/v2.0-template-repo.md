# Agency v2.0 - Templates

## Overview

Agency bundles templates in the `templates/` directory for quick project setup.

## Local Templates

Templates are stored in the `templates/` directory within the agency package:

```
agency/
└── templates/
    ├── basic/
    │   └── .agency/
    ├── api/
    │   └── .agency/
    └── fullstack/
        └── .agency/
```

### Available Templates

| Template | Description |
|----------|-------------|
| `basic` | Minimal setup with one coder agent |
| `api` | API project with backend and frontend agents |
| `fullstack` | Fullstack project with backend, frontend, and devops agents |

## Using Templates

```bash
# List available templates
agency templates

# Initialize with default (basic) template
agency init --dir ~/projects/myproject

# Initialize with specific template
agency init --dir ~/projects/myproject --template api

# Initialize with custom GitHub template
agency init --dir ~/projects/myproject --template https://github.com/user/repo/tree/main/my-template
```

## Custom Template Repository

You can use any GitHub repository as a template source:

```bash
# Using full URL
agency init --dir ~/projects/myproject \
  --template https://github.com/user/custom-templates/tree/main/my-template

# Using owner/repo format
agency init --dir ~/projects/myproject \
  --template user/custom-templates
```

## Creating Custom Templates

Create a template directory with a `.agency/` subdirectory:

```
my-template/
└── .agency/
    ├── config.yaml           # Required
    ├── manager.yaml          # Optional
    ├── agents.yaml           # Optional
    └── agents/              # Optional
        └── *.yaml
```

### Example Template

```bash
# Create template structure
mkdir -p my-template/.agency/agents

# config.yaml
cat > my-template/.agency/config.yaml << 'EOF'
project: my-template
shell: bash
EOF

# manager.yaml
cat > my-template/.agency/manager.yaml << 'EOF'
personality: |
  You are the project coordinator.

poll_interval: 30
EOF

# agents.yaml
cat > my-template/.agency/agents.yaml << 'EOF'
agents:
  - name: developer
    config: agents/developer.yaml
EOF

# Agent config
cat > my-template/.agency/agents/developer.yaml << 'EOF'
name: developer
personality: |
  You are a skilled developer.
EOF
```

## Caching

Downloaded templates are cached in `~/.cache/agency/templates/`:

```
~/.cache/agency/templates/
├── basic/                    # Local templates (copied)
└── github.com/
    └── user/
        └── custom-repo/
            └── my-template/   # Remote templates (downloaded)
```

### Cache Management

```bash
# Clear all cached templates
rm -rf ~/.cache/agency/templates/

# Bypass cache (use --refresh)
agency init --dir ~/projects/myproject --template <name> --refresh
```

## Template Format

### config.yaml

```yaml
project: <template-name>
shell: bash
```

### manager.yaml

```yaml
personality: |
  You are the project coordinator.

  ## Your Role
  - Manage tasks using `agency tasks` commands
  - Assign work to free agents
  - Review completed work

poll_interval: 30
auto_approve: false
```

### agents.yaml

```yaml
agents:
  - name: coder
    config: agents/coder.yaml
```

### agents/<name>.yaml

```yaml
name: coder
personality: |
  You are a coder agent.
```
